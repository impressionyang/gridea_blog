访问本站方法

  • 添加gridea部署的博客https://impressionyang.vercel.app/，同步发布内容

本站一共有三种方法可以访问到

  • 外国通道（github）：impressionyang.github.io
  • 国内通道（gitee）： impressionyang.gitee.io
  • 国内通道 （西部数码）：impressionyang.top

由于西部数码使用的是虚拟主机，而更新插件ftpsync有毛病了，所以更新全靠手动，更新会晚很多，请尽量使用git通道

使用哪种方法进行访问就看你自己的选择了。

文档大概分类

categories

  • 分为三种分类
  1. 日常类,categories=daily
  2. 编程类,categories=program
  3. 技巧类,categories=skill
  4. 培训学习类，catagories=learn

tag

  • 下面是主标签，每个帖子会发布多标签，一般为主标签+具体标签
  • 第一个标签（主标签）就是帖子的分类
  1. 日常类,tag=daily,...
  2. 编程类,tag=program,...
  3. 技巧类,tag=skill,...
  4. 培训学习类，tag=learn,华清远见farsight

文章查找

直接搜索关键词

在首页可以点击博客网站的顶部的搜索按钮进行关键词搜索

归档

在任意页面博客站顶部的文章->归档进行上传日期为索引的文章检索

标签和分类

在任意页面博客站顶部的文章->分类/标签进行以 分类/标签 为索引的文章检索
